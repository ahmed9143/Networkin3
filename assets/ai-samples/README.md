# صور الـ AI Detection في الهيرو

الكارت اللي بيظهر جنب كل Detection Box (Person / Car / Face / Plate) بيعرض دلوقتي
**رسومات SVG أصلية اتصممت خصيصًا لهوية الموقع** (مش صور حقيقية من الإنترنت)، عشان
نتجنب أي مشكلة حقوق نشر. الملفات موجودة في الفولدر ده:

| النوع    | الملفات                                  |
|----------|-------------------------------------------|
| Person   | `person-01.svg`, `person-02.svg`           |
| Car      | `car-01.svg`, `car-02.svg`                 |
| Face     | `face-01.svg` (شكل تجريدي/mesh، مش وش حقيقي)|
| Plate    | `plate-01.svg`, `plate-02.svg`              |

## لو عايز تستبدلها بصور حقيقية لاحقًا
1. لازم تكون صورك انت (أو مرخّصة قانونيًا ومدفوعة الرخصة) — عربية حقيقية، شخص
   حقيقي وافق على النشر، إلخ. متستخدمش صور فيها لوجو شركة تانية أو علامة مائية.
2. المقاس المثالي: نسبة 4:3، حجم خفيف (اعمل ضغط قبل الرفع).
3. الكود بيدور على امتداد `.svg` بالاسم بالظبط زي الجدول فوق. لو عايز تحط JPG بدالها،
   قولّي وهغيّرلك السطر في الكود يدور على `.jpg` بدل `.svg`.
4. لو الملف مش موجود، الكود مش هيبوّظ — هيرجع تلقائيًا لأيقونة SVG بسيطة.
